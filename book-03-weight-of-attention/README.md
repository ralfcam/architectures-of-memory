# Book 3 — The Weight of Attention

**The Architectures of Memory**, Book 3. *Is Retrieval the Same as Memory?*

Attention and the Transformer replace the recurrent hidden-state bottleneck with weighted access to the whole past. The book asks what that shift means technically, in organizations (search, RAG, compliance), and in philosophy (Meno, extended mind, relevance as construction).
