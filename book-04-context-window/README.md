# Book 4 — The Context Window

**The Architectures of Memory**, Book 4. *What Does Scale Actually Know?*

Large language models distribute memory across parameters and a finite context window. This volume treats parametric memory, in-context learning, hallucination, RAG, and emerging architectures — then regulated industries, governance gaps, and philosophical questions about pattern versus understanding.
