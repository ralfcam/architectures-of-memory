# Book 2 — The Forget Gate

**The Architectures of Memory**, Book 2. *What Should a Mind Choose to Keep?*

The LSTM and its forget gate are treated as a philosophical object: selective retention and forgetting as learned judgment. The book follows the technical arc from Hochreiter and Schmidhuber to practice and variants, then institutional memory, Borges, neuroscience of forgetting, and the ethics of machine memory.
